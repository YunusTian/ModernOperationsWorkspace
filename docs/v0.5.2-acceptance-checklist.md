## 1. Manifest `settingsSchema` 编译器

- [x] `Compile(raw)` 支持关键字子集：`type / properties / required / enum / default / description / title / format / secret / minimum / maximum / minLength / maxLength / pattern / items / additionalProperties`（[core/plugin/settings/schema.go](../core/plugin/settings/schema.go)）
- [x] 顶层强制 `type: object`；不允许其他值
- [x] 支持自定义关键字 `"secret": true`（叶子/嵌套皆可，含数组 items 内部）
- [x] `Validate(raw)` 输出稳定排序的 `[]Error`；覆盖 required / enum（含跨类型）/ 数字范围 / 字符串长度与 pattern / `additionalProperties=false`
- [x] `ApplyDefaults(raw)` 递归注入 default，present 字段不覆盖
- [x] `Redact(raw)` 把所有 secret 叶子替换成 `"***"`；数组元素独立处理
- [x] `Fields()` 返回 UI 稳定顺序（字典序 + 深度）
- [x] `SetPath(raw, path, value)` / `GetPath(raw, path)` 支持点号路径与自动创建中间层
- [x] `SecretPaths()` 输出所有 secret 叶子路径（数组用 `[]`）
- [x] 单测：Compile 空/非 object、Validate 五组约束、ApplyDefaults、Redact（含数组）、SetPath/GetPath、enum 混合类型、Fields 顺序 & 深度（[schema_test.go](../core/plugin/settings/schema_test.go)）
